重要提示

计算时一定要对齐数位，就像排队一样，个位站一列，十位站一列。计算过程中要特别注意进位和退位，可以在数字上方标记小数字帮助记忆。

例题一：加法计算

计算：458 + 376 = ?
1. 个位：8 + 6 = 14，写4进1
2. 十位：5 + 7 = 12，加上进位1得13，写3进1
3. 百位：4 + 3 = 7，加上进位1得8
4. 最终结果：458 + 376 = 834
答案：834

例题二：减法计算

计算：725 - 389 = ?
1. 个位：5-9不够减，向十位借1，15-9=6
2. 十位：2被借走1剩1，1-8不够减，向百位借1，11-8=3
3. 百位：7被借走1剩6，6-3=3
4. 最终结果：725 - 389 = 336
答案：336

例题三：带零的减法

计算：503 - 267 = ?
1. 个位：3-7不够减，向十位借1，但十位是0
2. 十位向百位借1，百位5变成4，十位0变成10
3. 十位再借1给个位，十位10变成9，个位3变成13
4. 个位：13-7=6
5. 十位：9-6=3
6. 百位：4-2=2
答案：236

例题四：时间计算

小明从家到学校需要25分钟，他7:40从家出发，几点到达学校？
1. 开始时间：7:40
2. 经过时间：25分钟
3. 计算：40分钟 + 25分钟 = 65分钟
4. 65分钟 = 1小时5分钟
5. 7时 + 1小时 = 8时
6. 40分 + 25分 = 65分 → 1小时5分
答案：8:05

例题五：时间差计算

学校运动会从上午9:15开始，到11:40结束，一共进行了多长时间？
1. 从9:15到11:15是2小时
2. 从11:15到11:40是25分钟
3. 所以总共：2小时 + 25分钟 = 2小时25分钟
答案：2小时25分钟
